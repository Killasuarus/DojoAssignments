What are Databases?

A structured set of data

Databases are a collection of information, they are used for creating, manipulating, and deleting data. There are many different types of databases and they can be classified in a basic approach of relational database or non-relational databse. Relationships means a set of our data us related to another section of our data. Meanwhile in a non-relational database there is not a common related section of our data. Our databases are put sections of data called schemas, inside each schema each data set is organized into tables, those tables contain columns and rows to house that particular set of data.
